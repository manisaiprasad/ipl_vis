# ipl_vis

# Installation

```
git clone https://github.com/manisaiprasad/ipl_vis
cd ipl_vis
python3 -m venv env
source env/bin/activate
pip install -r requriments.txt
export FLASK_APP=main.py
export FLASK_ENV=development
flask run -h 0.0.0.0
```
